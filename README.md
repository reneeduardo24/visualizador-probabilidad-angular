# Visualizador Interactivo de Probabilidad y Conteo
Sistema web realizado en Angular para visualizar operaciones de conjuntos (con diagramas de Venn) 
y técnicas de conteo (permutaciones y combinaciones) con gráficas interactivas.

## Tecnologías
- Angular 19
- TypeScript
- SCSS
- Venn.js
- Chart.js + ng2-charts
- MathJax / KaTeX

## Módulos principales
- Conjuntos (Venn, operaciones, independencia, disjunción)
- Técnicas de conteo (perm., comb., factoriales)
- Visualizaciones interactivas
